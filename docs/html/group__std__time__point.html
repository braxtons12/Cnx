StdTimePoint std_time_point_new(StdDuration time_since_epoch)

Constructs a new StdTimePoint representing the given time since the UNIX epoch, assuming the same precision as std_system_clock

Parameters
time_since_epoch - The time since the UNIX epoch. The precision of the StdTimePoint will be the period of this StdDuration
Returns a StdTimePoint representing the given time since the epoch

Note

if time_since_epoch has a different period than the precision of std_system_clock, it will be converted to the precision of std_system_clock
